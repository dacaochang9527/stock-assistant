# A 股中短线选股助手实施计划

## Phase 1：离线扫描版

- 建立项目骨架；
- 接入日线数据源；
- 实现基础指标：MA、EXPMA、成交量均值、涨停判断；
- 实现风险过滤：ST、北交所、低成交额、低价股、新股；
- 实现屠龙 D1/D2 识别；
- 实现波段候选识别；
- 生成 Markdown 日报。

## Phase 2：屠龙回测版

- 读取历史日线；
- 按 D1/D2/D3/D4/D5 状态回放；
- 输出胜率、平均收益、最大回撤、盈亏比；
- 单独验证：
  - 天安新材 603725，2023-07-28；
  - 佳力图 603912，2022-12-29。

## Phase 3：盘中提醒版

- 读取前一日 D2 合格候选；
- 盘中轮询价格；
- 触发 D3 水下观察、跌破支撑、D4 高开止盈、D5 到期提醒；
- 防重复提醒。

## Phase 4：个人复盘版

- 每次提醒记录结果；
- 支持人工标记有效/无效/误报；
- 每周生成策略表现报告。
